The code in SMOTE.py performs process of creating synthetic data on an imbalanced dataset where there are some minority values in a column of dataset to oversample those minority values SMOTE is  been used for example in fraud detection dataset if there is a column which contains 95 % values as 'Fraud' and 5% as 'Normal' Model always predicts 'Fraud' This problem arises dut to output column contains majority values as 'Fraud' and minority values as 'Normal' So model becomes habitual to print 'Fraud' So to avoid this problem We increase values 'Normal' Synthetically using SMOTE so model gives correct predication according to logic trained to him.
